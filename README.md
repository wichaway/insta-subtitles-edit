# עורך וידאו + כתוביות אוטומטיות

עורך וידאו שרץ כולו בדפדפן, בלי שרת: מעלים כמה סרטונים, הם מתחברים אוטומטית עם מעבר חלק (crossfade), אפשר לחתוך/לסדר מחדש ידנית, ליצור כתוביות אוטומטית באמצעות זיהוי דיבור (Whisper, רץ מקומית בדפדפן), לתקן תזמון וטקסט ברשימת כתוביות, לעצב את מראה הכתוביות (גופן, גודל, צבע, מיקום), ולייצא סרטון מוכן (WebM או MP4).

## הרצה מקומית

```bash
npm install
npm run dev
```

## איך זה עובד

- **חיבור קליפים**: `src/lib/timeline.ts` מסדר את הקליפים על ציר זמן משותף עם חפיפה (crossfade) בין כל שניים.
- **תצוגה מקדימה**: `src/lib/compositor.ts` מרנדר כל פריים ל-`<canvas>` — מערבב שני קליפים בזמן המעבר, ומצייר את הכתובית הפעילה מעליהם (`src/lib/subtitleRender.ts`).
- **זיהוי דיבור אוטומטי**: `src/lib/extractAudio.ts` ממזג את האודיו של כל הקליפים, ו-`src/lib/whisperWorker.ts` מריץ מודל Whisper (דרך `@xenova/transformers`) ב-Web Worker כדי לא לחסום את הממשק. המודל והמשקלים שלו נטענים מ-Hugging Face בפעם הראשונה שמשתמשים בזיהוי הדיבור.
- **ייצוא**: `src/lib/export.ts` מקליט את ה-canvas והאודיו בזמן אמת (MediaRecorder) ל-WebM; אם נבחרה המרה ל-MP4, `src/lib/transcodeMp4.ts` ממיר באמצעות `@ffmpeg/ffmpeg` (טוען את הליבה מ-unpkg בפעם הראשונה).

## הערות

- זיהוי הדיבור וההמרה ל-MP4 דורשים גישה לאינטרנט בפעם הראשונה (להורדת מודל/ליבת ffmpeg מ-CDN ציבורי); אחרי הטעינה הראשונה הם נשמרים במטמון הדפדפן.
- התזמון האוטומטי של הכתוביות מבוסס זיהוי דיבור ועלול לסטות מעט, בעיקר סביב מעברים בין קליפים — ניתן לתקן כל כתובית ברשימה למטה.
